# AWS Security Assessment Checklist

Checklist to validate AWS environment configuration and controls.

---

## Logging & Detection

✔ CloudTrail across regions  
✔ VPC Flow Logs enabled  
✔ GuardDuty enabled  
✔ Config rules active

---

## IAM Security

✔ MFA enabled on root and admin users  
✔ Least privilege policies enforced  
✔ Access keys rotated

---

## Networking

✔ NACLs and SG rules reviewed quarterly  
✔ Public S3 access restricted  
✔ Private subnets used for sensitive services

---

## Monitoring

✔ CloudWatch alarms configured  
✔ SIEM integration completed
