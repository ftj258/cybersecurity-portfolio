# Azure Security Assessment Checklist

Checklist to validate Azure environment security.

---

## Identity & Access

✔ Azure AD PIM enabled  
✔ Conditional Access policies applied  
✔ MFA enforced enterprise-wide

---

## Logging & Monitoring

✔ Azure Monitor logs streaming  
✔ Sentinel rules enabled  
✔ Activity log alerts configured

---

## Networking

✔ NSGs applied to all subnets  
✔ Private endpoints reviewed  
✔ Firewall rules documented

---

## Compliance

✔ Azure Policy definition enforcement  
✔ Resource tagging compliance
