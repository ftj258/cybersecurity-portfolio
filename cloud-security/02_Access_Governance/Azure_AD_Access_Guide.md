# Azure AD Access Governance Guide

**Objective:** Best practices to secure Azure Active Directory identities.

---

## Principles

• Enforce Conditional Access  
• Ensure Secure Defaults  
• Require Phishing-resistant MFA  
• Use Privileged Identity Management (PIM)

---

## Monitoring

• Audit sign-ins  
• Risky sign-in alerts  
• MFA challenge logs  
• OAuth trace review

---

## Remediation

• Disable risky access policies  
• Audit legacy authentication  
• Revoke stale app permissions

---

## Skills Demonstrated

• Azure identity governance  
• RBAC implementation  
• Conditional access tuning
