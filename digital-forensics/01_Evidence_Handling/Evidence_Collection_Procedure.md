# Evidence Collection Procedure

## Objective

Provide standardized process for acquiring digital evidence from endpoints and cloud systems.

---

## Endpoint Collection

- Capture volatile memory (if required)
- Create forensic disk image
- Collect system logs
- Export browser artifacts
- Preserve registry hives

---

## Email Evidence Collection

- Export full email headers
- Preserve original message format (.eml/.msg)
- Extract attachment hashes
- Capture authentication results (SPF/DKIM/DMARC)

---

## Cloud Evidence Collection

- Export audit logs (AWS CloudTrail / Azure Activity Logs)
- Capture IAM role assumptions
- Document session tokens
- Preserve security group configuration

---

## Hashing & Validation

- Generate SHA256 hash
- Verify post-transfer hash
- Document all transfer steps

---

## Skills Demonstrated

- Multi-environment evidence acquisition
- Cloud forensic log handling
- Technical documentation precision
